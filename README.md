This project predicts which employees are at risk of leaving the company and uses SHAP (SHapley Additive exPlanations) to explain why identifying the specific factors driving each individual employee's risk score, rather than just outputting a single probability.

Dataset

IBM HR Analytics Employee Attrition dataset (downloaded from kaggle)

Tech Stack - 
Python, Pandas
Scikit-learn / Random Forest (model training)
SHAP (explainability)
Plotly (visualizations)
